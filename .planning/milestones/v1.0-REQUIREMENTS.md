## v1 Requirements — All Delivered

### Webcam & Detection

- [x] **CAM-01**: User can enable webcam tracking via "Use ZoomMe" checkbox in popup — Phase 1 ✓
- [x] **CAM-02**: Enabled state is per-session only — always starts unchecked on browser restart — Phase 1 ✓
- [x] **CAM-03**: Webcam fully stops (stream released) when checkbox is unchecked — Phase 1 ✓
- [x] **CAM-04**: Visual indicator in popup shows when webcam is active — Phase 1 ✓
- [x] **CAM-05**: Normal distance auto-calibrated when user first enables (capture baseline) — Phase 2 ✓

### Zoom Control

- [x] **ZOOM-01**: Page zoom adjusts based on head distance — closer = zoom out, farther = return to 100% — Phase 3 ✓
- [x] **ZOOM-02**: Zoom uses Chrome's tabs.setZoom() API (native browser zoom) — Phase 3 ✓
- [x] **ZOOM-03**: Zoom resets to 100% when user disables tracking — Phase 3 ✓
- [x] **ZOOM-04**: EMA smoothing + dead zone prevents jitter during normal use — Phase 3 ✓
- [x] **ZOOM-05**: Slider in popup controls max zoom range — Phase 4 ✓
- [x] **ZOOM-06**: Head distance measured at ~1 second intervals (not every frame) — Phase 2 ✓

### Site Exclusion

- [x] **EXCL-01**: User can toggle exclude current site via extension icon/popup — Phase 4 ✓
- [x] **EXCL-02**: Excluded sites list visible in popup with remove capability — Phase 4 ✓
- [x] **EXCL-03**: Excluded sites persist across sessions via chrome.storage — Phase 4 ✓

### Settings

- [x] **SET-01**: Slider position persists across popup close via chrome.storage — Phase 4 ✓
- [x] **SET-02**: Excluded sites persist across sessions via chrome.storage — Phase 4 ✓
